# Proyecto Intermodular SMR: Infraestructura de Red para Interiorismo

**Autor:** Alejandro Cañas Moreno
**Ciclo:** Sistemas Microinformáticos y Redes (SMR)

## Descripción
Este repositorio contiene la documentación y los archivos de configuración del proyecto intermodular. El objetivo ha sido diseñar y simular una infraestructura de red corporativa para una empresa de interiorismo, incluyendo:
- Virtualización de servidores y estaciones de trabajo.
- Configuración de servicios de red (Active Directory, DNS, DHCP).
- Gestión de permisos y recursos compartidos.

## Estructura del Proyecto
- `/memoria`: Documentación técnica y memoria del proyecto.
- `/scripts`: Scripts de configuración y automatización.
- `/config`: Archivos de configuración de red y entornos.

## Instrucciones
Para revisar el proyecto, consulta la memoria técnica alojada en la carpeta correspondiente.
